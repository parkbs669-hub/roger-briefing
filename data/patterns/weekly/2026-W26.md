# 2026-W26 주간 패턴 분석 보고서

> **분석 기준일**: 2026-06-26 | **누적 6주차** (W21 베이스라인 대비 패턴 검증 주차)

---

## 1. 이번 주 효과적이었던 것

### ✅ "포스터 지참 + 현장 즉시 전달"이 지회 설득의 실질 열쇠임을 확인

- 광명시 피드백(W21)에서 이미 포착한 패턴 — **"중앙 공문·포스터 없이는 지회가 움직이지 못한다"** — 를 이번 주 대구지회 방문에 선제 적용
- 인터넷에서 포스터 출력 후 직접 전달 → **구두 미팅 약속(7월 7일) 확보**로 이어짐
- **핵심 발견**: W21에서 "병목으로 기록"되었던 지회 자체 제작 여력 부재 문제를, 담당자가 직접 출력물을 들고 가는 방식으로 우회 돌파. 완성 자료 제공이 지회 움직임의 전제조건임이 **2개 지역(광명·대구)에서 반복 검증**됨

### ✅ 지역 내부 안건화 루트 발견 — "7월 2일 각 지회 미팅" 레버리지

- 대구지회 조남인 총무처장이 **7월 2일 대구 각 지회 미팅 안건으로 자발적 전달 의사 표명**
- 담당자가 단순 수신자가 아닌 **내부 전파자(internal champion)**로 전환된 첫 사례
- 노인정 1,860개라는 규모 정보 확보 → 대구 지역 확산 잠재력 수치화 가능

> **W21 패턴 강화**: "내부 챔피언을 통한 진입이 수용 속도를 높인다"는 기존 패턴이, 이번에는 기관 외부 인맥이 아닌 **기관 내부 담당자 자신이 챔피언화**되는 형태로 진화 확인

### ✅ 인협-노인회 연계 가능성 구조화 시작

- 인구보건협회 방문에서 **노인회에서 건협·인협 함께 진행 가능 여부** 검토 방향 확인
- 기관-병원 비연계 방향 공감 → 역할 분리 프레임 정립
- 단일 채널이 아닌 **인협+노인회+건협 삼각 협력 구조** 설계가 진전됨

## 2. 효과 없었거나 개선 필요한 것

### ❌ 천안 서북구청 — 중간 창구 의존 리스크 현실화

- 이원 대표가 아직 미팅을 잡지 못하고 있음 → **W21부터 지속된 정체가 5주 이상 지속**
- 구조적 문제: 구청장 직접 접근 루트가 이원 대표 단일 창구에 집중 → 창구가 막히면 전체 프로젝트가 멈춤
- **개선 필요**: 이원 대표 외 **보조 진입 루트**(서북구청 보건소 주무관, 담당 팀장 직접 접촉 등) 병행 개발 시급. 단일 챔피언 의존은 W21에서도 경고 신호였으나 아직 미해결

### ❌ "구두 약속"이 여전히 다음 단계로 전환되지 못하는 패턴 반복

- 대구지회: 7월 7일 미팅 약속 → **긍정적이나, 이 약속이 "본부 담당자 면담"으로 이어질 구체적 준비(어젠다, 전달 자료, 참석자 확정)가 부재**
- W21에서 지적한 "공감 이후 약속의 물질화 실패" 패턴이 W26에서도 반복
- **개선 필요**: 7월 7일 미팅 전까지 ① 면담 어젠다 문서화 ② 본부 담당자 이름·직함 확인 ③ 전달할 제안서 버전 확정을 완료해야 "구두 약속→실질 진전" 전환 가능

### ❌ 미완료 후속 조치가 14개 이상 — 실행 과부하 징후

- 이번 주 미완료 태스크: 정책청 NIP 확인, 광진구 보건소 접촉, MSD 모니터링, 조례 검토, 예산 산정 등 **성격이 다른 14개 이상의 태스크가 동시 미완료 상태**
- 전략적 우선순위 없이 모든 항목이 동등하게 쌓이는 구조 → 실행 집중도 저하
- **개선 필요**: 주간 시작 시 "이번 주 반드시 끝낼 3개"를 명시적으로 지정하는 루틴 필요

## 3. 고객 반응 패턴 발견

### 🔍 유형별 반응 업데이트 (W26 신규 데이터 반영)

| 고객 유형 | 반응한 것 | 반응하지 않은 것 | W26 신규 발견 |
|---|---|---|---|
| **노인회 지회 간부** (대구 조남인 총무처장) | 완성된 포스터 현물 + MOU 인지 전제 설명 | 구두 설명만의 미팅 | **내부 미팅 안건화 자발 제안** — 자료가 있으면 스스로 전파자가 됨 |
| **노인회 지회** (광명시, W21) | 중앙 파일 수령 후 경로당 배포 | 자체 기획·제작 | 턴키 솔루션 확인 (W21과 동일, 패턴 확정) |
| **인협 담당자** (곽동희님) | 기관-병원 비연계 프레임 + 다기관 연계 가능성 | 독자 진행 방식 | 노인회·건협 연계 공동 추진 방향 공감 |
| **지자체 공무원** (제주 팀장, W21) | 과학적 홍보 + QR코드 등 실용 도구 | 기존 포스터 방식 | (W21 확인, W26 추가 데이터 없음) |

### 🔍 이번 주 핵심 신규 패턴

> **"자료를 손에 쥐어주면, 지회 담당자는 자발적 전파자가 된다"**
>
> 광명(W21)에서는 "전달 가능 의향"을 확인하는 데 그쳤다면, 대구(W26)에서는 담당자가 **7월 2일 내부 지회 미팅 안건**으로 직접 올리겠다고 제안 → 자료 제공이 단순 배포를 넘어 **내부 의제 설정**까지 유발함

### 📊 프로젝트별 상태 진단

| 프로젝트 | 현재 상태 | 병목 | 돌파구 |
|---|---|---|---|
| **대한노인회 대구지회** | 🟡 모멘텀 확보 (7/7 약속) | 본부 연결 여부 미확인 | 7/2 지회 미팅 후속 확인 → 7/7 본부 면담 어젠다 선제 준비 |
| **천안 서북구청** | 🔴 5주 이상 정체 | 이원 대표 단일 창구 | 보조 진입 루트(보건소 직접 접촉) 병행 |
| **인협_협력** | 🟡 40% 정체 | 경렬님 동행 방문 미실행 | 이번 주 내 방문 날짜 확정 필요 |
| **대구_코로나** | 🟠 50% (변동 큰 편) | 진행률 편차 원인 불명확 | 다음 단계 액션 명시화 필요 |

### 📊 구조적 진행 패턴 (W21→W26 공통 관찰)

**공감 단계(30%) → 약속 단계(50%) 전환 실패**가 가장 반복적인 병목으로, 이 구간을 넘긴 프로젝트가 아직 없음.

> **패턴 가설**: 공감 이후 "다음 행동의 주체"가 고객에게 넘어가는 순간 속도가 급격히 저하됨. 담당자가 직접 다음 스텝을 보유하고 주도할 때(예: 포스터 전달 → 7/7 약속)만 진전이 발생

## 5. 다음 주 적용할 전략 (패턴 기반)

### 🎯 Priority 1. 7월 7일 대구 본부 면담 — "구두 약속의 물질화" 집중

- **할 일**: 면담 전 ① 어젠다 1페이지 문서 작성 ② 본부 담당자 이름·직책 사전 확인 ③ 대구 지회 노인정 1,860개 규모 기반 제안 수치 업데이트
- **클로징 목표**: 면담 당일 "다음 회의 날짜 또는 전달 문서 종류" 중 하나 이상 확정
- **근거**: W21·W26 반복 패턴 — "물질화 없는 공감은 정체로 귀결"

### 🎯 Priority 2. 천안 서북구청 — 보조 루트 개설 시도

- **할 일**: 이원 대표 진행 상황 주 1회 확인 + 서북구청 보건소 담당 주무관 직접 콜드 접촉 병행
- **근거**: 단일 창구 의존 5주 → 리스크 헤지 필요

### 🎯 Priority 3. 인협+노인회+건협 삼각 연계 — 경렬님 동행 방문 날짜 확정

- **할 일**: 이번 주 내 경렬님 일정 확인 → 방문 날짜 캘린더 고정
- **근거**: "이번 주" 예정이 W26에서 실행되지 않은 채 넘어감 → 날짜 없는 계획은 실행되지 않는 패턴

### 🎯 Priority 4. 미완료 태스크 트리아지

- 14개 미완료 태스크 중 **이번 주 반드시 3개, 다음 달로 이월 가능 5개, 폐기/위임 가능 항목** 분류
- **근거**: 실행 과부하가 핵심 프로젝트 집중도를 떨어뜨리는 구조적 위험
