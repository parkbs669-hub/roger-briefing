# 2026-W25 주간 패턴 분석 보고서

> **분석 기준일**: 2026-06-19 | **W21 대비 4주 누적 비교 분석**

## 1. 이번 주 효과적이었던 것

### ✅ "인협-노인회 연계" 복합 채널 제안이 새로운 돌파구 후보로 부상
- 곽동희님(인구보건협회) 방문에서 **"노인회에서 건협·인협 함께 진행 가능 여부 검토"** 라는 복합 파트너십 구조 확인
- 단일 기관 공략이 아닌 **노인회(접근 채널) × 건협·인협(접종 실행 채널)** 이중 구조를 동시에 가동하는 모델이 구체화됨
- **W21 패턴과 비교**: W21에서 "멀티채널 포위망"이 기반 마련 수준이었다면, 이번 주는 채널 간 **연결 고리**가 실제로 논의된 진전

> **신규 패턴**: 단순 멀티채널이 아닌 "채널 간 역할 분담(접근 vs 실행)"이 명확해질 때 기관의 실행 의지가 높아짐

### ✅ 지역 밀착 인맥(이원 대표 - 천안 구청장 동창)이 여전히 가장 강력한 진입 수단
- W21에서도 동일 패턴 확인, **이번 주에도 재확인됨** → 제안서 전달 + 공감 확보까지 1회 미팅으로 완료
- **W21과의 차이**: W21은 패턴 발견 단계, **W25 현재는 이 채널이 구조적으로 재현 가능한 루틴임을 확증**

### ✅ COVID 마케팅에서 "접종 연장 → 재고 소진 필요성" 프레임 실전 투입
- 지자체 상위(주무관)에게는 "파트너십·접종률 개선 기여" 메시지
- 지자체 하위(보건소)에게는 "재고 효율 소진 필요성" 메시지
- **타겟 레벨별 메시지 분리 전략**이 실제 현장에서 가동되기 시작 → 신규 발견 패턴

## 2. 효과 없었거나 개선 필요한 것

### ❌ 대한노인회 중앙-지회 단절 문제: W21에서 이미 발견, W25에서도 미해결로 반복
- 광명시 지회 방문 시 **협업 내용 전혀 미인지** → W21에서 이미 동일 문제 기록
- **4주가 지났음에도 구조적 해결 없이 같은 벽에 재충돌** → 이것은 단순 실행 오류가 아닌 **프로세스 설계 실패**
- 중앙회 공문 발송 확인 → 지회 방문의 선행 조건화가 여전히 미적용

> **경고 패턴**: W21 → W25 동안 동일 문제가 반복된다는 것은 "알고 있지만 고치지 못하는 병목"으로 고착화 위험

### ❌ 6개 프로젝트 전체 30% 4주 연속 정체 — 패턴이 구조화됨
- 대한노인회(3주), 수성구청(2주), 천안 서북구청(3주), 제주도청(2주), 인협(2주), 대구 코로나(2주)
- **공통 패턴**: 모두 "공감" 단계에서 멈춤. "좋다" → "다음"으로 전환하는 **클로징 루틴 부재**
- 데일리 브리핑에서 매일 **"공감한 결정권자가 실무자를 직접 연결해주는 장면 1건"** 목표를 설정했으나 달성 여부 불확인

### ❌ 홍보 자료 공급 병목 — 지회 수준에서 실행 불가 상태 장기화
- 광명시 노인회: 포스터·안내문 자체 제작 불가, 중앙 파일 공급 시 배포 가능
- **요청 사항(@Kim Jae-Hee, @Lim Dong-Eun)이 발행됐으나 완료 여부 미확인**
- 지역별 건강관리협회 주소·연락처 포함 포스터 제작이 선행되지 않으면 지회 현장 활동 전체가 정지 상태

### ❌ 제주도청 — QR코드 홍보 제안 후 구체적 다음 단계 부재
- 팀장이 자발적으로 QR코드 활용 제안(W21 패턴: 자발적 아이디어 = 오너십 신호)
- 그러나 **QR코드 콘텐츠 기획, 제작 일정, 배포 방법** 등 실행 구체화 미진행

## 3. 고객 반응 패턴 발견

### 🔍 이번 주 신규 발견 반응 유형

| 고객 유형 | 이번 주 반응한 것 | 반응하지 않은 것 | W21 대비 신규 발견 |
|-----------|-----------------|-----------------|-------------------|
| **노인회 지회**(광명시) | 중앙 공문 이후 설명 시 공감 | 자체 기획·제작 요청 | 공감하지만 **실행 자원 제로** → 본사 턴키 공급 없이는 영구 정체 |
| **인협**(곽동희님) | 기관 간 연계 모델(노인회+건협+인협) | 단독 기관 역할 요청 | **복합 파트너십 구조**에서 적극성 증가 |
| **지자체 감염팀장**(제주) | 과학적 홍보 필요성, 실용 도구(QR) | 일방적 자료 배포 | *(W21과 동일 패턴 재확인)* |
| **COVID 지자체 주무관** | 파트너십 메시지 + 접종률 기여 강조 | 제품 스펙 중심 접근 | **레벨별 메시지 분리**가 유효함을 확인 |
| **지자체 보건소** | 재고 소진 필요성 프레임 | 추상적 공중보건 메시지 | 실무 압박(재고) 언어가 보건소 레벨에서는 더 유효 |

### 🔍 이번 주 핵심 반응 패턴
> **"우리가 단독으로 해야 하는 일"이라고 느끼면 멈추고, "여러 기관이 함께 하는 구조"라고 느끼면 움직인다**
> - 노인회 지회: 혼자 하면 불가 → 중앙+건협+인협이 지원하면 가능
> - 인협: 단독 역할 부담 → 노인회 연계 시 적극성 증가
> - **시사점**: 제안 프레임을 "화이자 vs 기관 1:1"이 아닌 "생태계 구축"으로 설계할 것

## 4. 프로젝트 진행 패턴

### 📊 진행률 정체 분석

```

```

### 🔍 병목 지점 구조 분석

**1단계 (0→30%)**: 접촉 → 공감 확보
- ✅ **현재 잘 되는 구간** — 인맥 레버리지, 과학적 근거 프레임으로 대부분 통과

**2단계 (30→60%)**: 공감 → 실무자 연결 + 구체적 약속
- 🔴 **전 프로젝트 공통 병목** — 결정권자 공감 후 "다음 단계 약속의 물질화" 실패
- 담당자 지정, 날짜 확정, 요청 문서 종류 합의 등이 미팅 종료 전 이루어지지 않음

**3단계 (60→100%)**: 실행 → 접종 프로그램 가동
- ⬜ 아직 진입한 프로젝트 없음

### 🔍 돌파구 후보
- **인협-노인회-건협 복합 연계 모델**: 단일 기관 30% 정체를 돌파할 수 있는 구조적 변화 가능성
- **재중약품 + 천안 이원 대표 루트**: 3자 협력(제약사 파트너 포함)이 지자체 신뢰도를 높일 수 있음
- **노인복지관 연계**: 이번 주 새로 언급된 채널 — 노인회와 별도의 현장 실행 채널로 활용 가능성
