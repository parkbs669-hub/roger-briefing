# 2026-W23 주간 패턴 분석 보고서

> **분석 기준일**: 2026-06-05 | **누적 3주차** (W21 베이스라인 → W23 패턴 강화·수정)

## 1. 이번 주 효과적이었던 것

### ✅ 선거 직후 공백기를 '선점 포지셔닝' 시간으로 활용
- 6·3 지방선거 당선인 확정 직후, 지자체 담당자들이 조직 개편 불안 심리에 진입하는 타이밍에 접촉 유지
- 이 시기 "새 담당자가 와도 이미 논의 중인 사업"으로 기록을 남겨두는 것이 핵심 — 데일리 브리핑에서도 이를 "골든타임"으로 반복 강조
- **새로 발견된 패턴**: 선거·인사 이슈는 통상 영업 공백기로 취급되지만, 반대로 **후임 담당자가 인수인계받을 '기존 협의 사항' 목록에 들어가는 것**이 훨씬 유리한 포지셔닝임

### ✅ 인협·노인회 연계 시 '기존 채널 묶음' 전략이 실질 진전 창출
- 인구보건협회 방문에서 "노인회에서 건협·인협 함께 진행 가능 여부"로 논의가 확장
- 기관-병원 연계 배제 방향 합의 → **접종 주체를 단순화**함으로써 실행 가능성에 대한 공감대 형성
- W21 대비 새로운 점: 단순 제안이 아닌 **실행 조건(기관·병원 연계 제외)을 상호 합의**한 첫 사례 → 협의 구체화의 신호

### ✅ COVID 라인에서 수요조사 데이터 확보로 입찰 대응 선제 포지셔닝
- 대구 주미건 주무관을 통해 8개 보건소 수요량 취합 완료 (5월 19일 기준)
- 마케팅팀과 타임라인 공유(질병청 입찰·수요조사 일정)가 사전 조율된 상태 → 현장-본사 연동이 이번 주 가장 매끄럽게 작동한 사례

## 2. 효과 없었거나 개선 필요한 것

### ❌ 진행률 30% 정체가 3~4주째 지속 — "공감의 함정"에서 탈출 실패
- 천안 서북구청: 3주째 추적, 진행률 30% 고정
- 대한노인회·수성구청·제주도청: 2~3주 추적, 동일하게 30% 정체
- **W21에서 이미 진단했던 문제가 이번 주에도 반복됨** → 미팅 마감 클로징 루틴 미적용이 구조적으로 굳어지는 중
- **이번 주 새로 추가된 위험 신호**: 30%에서 3주 이상 멈추면 선거 이후 담당자 교체 시 **처음부터 재설명해야 하는 리셋 리스크** 발생

### ❌ 대한노인회 지회 정보 단절 문제 — 해결책 제시됐으나 실행 여부 불명확
- W21에서 진단: 중앙회 → 지회 전달 체계 부재
- 이번 주 경렬님 동반 방문 예정(대한노인회)으로 진전 시도했으나 방문 결과 데이터 없음
- **미완료 후속조치 목록에 "대한노인회 협력 회의"가 여전히 등재** → 실행 전환 지연

### ❌ 제주도청 — 동시접종 이슈 해결 없이 논의 진전 차단
- "동시접종 안 되는 기관은 워킹하지 않겠다" 방향이 이미 확인됐음에도
- **회사 차원의 설문조사 필요** 언급 → 현장 담당자 수준에서 해결 불가, 본사·마케팅 에스컬레이션 필요
- W21 진단(초기 접촉 시 동시접종 스크리닝)이 실행되지 않은 채 프로젝트가 계속 추적 중인 상태

### ❌ 미완료 후속조치 항목 과잉 — 우선순위 없는 리스트가 실행력 분산
- 이번 주 미완료 후속조치 21개 등재, 그러나 담당자 미지정 항목 다수
- "50~64세 민간시장 진입 전략 수립", "Capvaxive 의료진 교육 자료 준비" 등 전략적 과제가 운영성 과제와 동일 목록에 혼재 → 실행 순서 불명확

## 3. 고객 반응 패턴 발견

### 🔍 이번 주 새로 발견·강화된 유형별 반응

| 고객 유형 | 새로 확인된 반응 | 작동하지 않은 것 | 이번 주 특이점 |
|-----------|----------------|-----------------|---------------|
| **지자체 감염 담당** (제주 팀장) | "회사 차원 설문조사" 요청 → 데이터 기반 해결책 요구 | 포스터 등 단방향 홍보물 | 본인이 직접 솔루션 구체화 요청 → 공동 문제해결 모드 진입 신호 |
| **기관 실무자** (인협 곽동희님) | 기관-병원 연계 제외 방향 합의 → 단순화된 실행 구조 선호 | 복잡한 다기관 연계 모델 | 실행 부담 최소화 프레임에 공감 |
| **지역 네트워크 보유 대표** (이원 대표) | 재중약품 동반 구조로 이미 진행 중 → 협력사 포함 시 추진력 유지 | 단독 제안 | 로컬 파트너 포함이 실행 속도 높임 |
| **노인회 지회** (광명시) | W21 패턴 재확인: 완성 자료 제공 시 배포 가능 | 자체 기획·온라인 활동 | 신규 없음 — 이전 패턴 그대로 고착 |
| **보건소 주무관** (대구 주미건) | 수요조사 데이터 취합 협조 — 질병청 지시 기반 행정 절차 준수 | 선제적 재고 소진 제안 | COVID 라인은 지자체 자율 의사결정 여지 낮음 → 질병청 연동 타이밍이 핵심 |

### 🔍 이번 주 핵심 반응 패턴 (신규 발견)
> **"설문조사·데이터"가 공무원 설득의 새로운 진입 경로로 부상**
> - 제주 팀장: "접종이 이루어지지 않는 부분에 대해 회사 차원의 설문조사 필요" → 화이자가 데이터 생산자 역할을 맡으면 공무원이 '근거 기반 정책 추진자'로 포지셔닝 가능
> - 기존 패턴(전례·레퍼런스 제시)과 결합하면: **기존 데이터(전례) + 신규 데이터(설문)** 이중 근거 구조가 가장 강력한 공무원 설득 프레임

## 5. 다음 주 적용할 전략 (패턴 기반)

### 🎯 최우선 과제: 30% 정체 → 50% 전환 — 프로젝트 2개 이상

**적용 전술**:

1. **클로징 루틴 즉시 도입** (W21 진단 이후 2주 미적용 → 이번 주부터 강제 실행)
   - 미팅 종료 전 체크리스트: ① 다음 미팅 날짜 OR ② 전달할 문서 종류 OR ③ 담당자 이름
   - 셋 중 하나도 없으면 미팅 종료하지 않는 원칙 설정

2. **제주도청 설문조사 — 1주 내 초안 제시로 모멘텀 유지**
   - 팀장이 직접 요청한 항목 → 화이자가 1주 내 "접종 장벽 설문 초안"을 들고 재방문
   - 이것이 30% → 50% 전환의 가장 명확한 경로

3. **선거 후 인사 공백기 활용 — 기존 협의 문서화 긴급 진행**
   - 천안·수성구청·대한노인회: 지금까지 논의 내용을 1페이지 요약 메모로 정리 → 담당자에게 "지금까지 협의한 내용 확인용"으로 발송
   - 후임 담당자 인수인계 자료에 화이자 협의 내용이 포함되도록 선점

4. **동시접종 이슈 — 본사 에스컬레이션 공식 요청**
   - 제주도청 케이스를 근거로 "동시접종 가능 기관 리스트 또는 가이드라인" 본사·마케팅 팀에 정식 요청
   - 현장에서 해결 불가능한 장벽을 더 이상 개인 영업력으로 돌파 시도하지 않음

5. **인협 + 노인회 연계 — 경렬님 방문 결과 이번 주 내 공유·다음 단계 확정**
   - 방문 결과를 팀 내 공유하고 "노인복지관 연계 실행 조건" 문서화
